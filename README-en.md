### Usage

```
In macOS / Linux:

⌘/win + G - search selected text with different search engine( Google, Yahoo, Bing, Baidu ... )

⌘/win + E - translate selected text

⌘/win + B - jump to selected link

**Note**

Shortcuts are customizable in `Options` page
```
